Описание плагина "Skmapsimple. Простая карта от google"
тема на livestreet.ru - http://livestreet.ru/blog/addons/15821.html

Плагин дает возмоность прикреплять к топикам простую google карту с одним маркером.

Можно настроить:
- видимость карт для незарегистрированных пользователей
- центр карты по умолчанию при создании топика

Лицензия:
